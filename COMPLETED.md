## 🚀 Getting Started

### 1. Install Dependencies
```bash
pnpm install
```

### 2. Configure Backend URL
```bash
cp .env.example .env.local
# Edit .env.local if backend is not at http://localhost:5000
```

### 3. Start Development Server
```bash
pnpm dev
```

### 4. Open in Browser
Navigate to `http://localhost:3000`
